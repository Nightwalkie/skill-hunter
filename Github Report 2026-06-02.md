Currently provides meridian-visa — a skill that walks agents through the complete visa readiness loop: look up corridor rules, file applications, fill trip details and documents, run readiness checks, dispatch the Meridian Computer to fill consulate portals, book appointments, and record outcomes. Anonymous tools work without login; user-scoped tools prompt for OAuth. Installs via npx into Claude Code, Cursor, Codex, or Goose. Complements the Meridian MCP server by adding workflow knowledge (when and in what order to use the tools).
